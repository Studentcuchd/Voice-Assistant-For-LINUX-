# Single Python installation file for the entire project.
#
# Install command:
#   python -m pip install -r requirements.txt
#
# Non-pip Linux system packages still required for full features:
#   Debian/Ubuntu:
#     sudo apt update
#     sudo apt install -y portaudio19-dev python3-pyaudio xdotool espeak-ng speech-dispatcher
#
#   Fedora:
#     sudo dnf install -y portaudio-devel python3-pyaudio xdotool espeak-ng speech-dispatcher
#
#   Arch:
#     sudo pacman -Sy --noconfirm portaudio python-pyaudio xdotool espeak-ng speech-dispatcher
#
# Notes:
# - SQLite is built into Python; no pip package required.
# - TTS uses system tools (spd-say/espeak), not Python packages.
